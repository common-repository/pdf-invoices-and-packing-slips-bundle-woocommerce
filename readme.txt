== Description ==

**Woocommerce Pdf Invoices and Packing Slips Bundle** using to setup PDF invoice and packing slip automatically generate when order place in [Woocommerce](https://en.wikipedia.org/wiki/WooCommerce). So Seller should not need to create PDF invoice manually. **Woocommerce pdf invoices packing slips delivery notes and shipping labels** all writen in **Woocommerce Invoice Pdf**.

> **Features**
>
> * Option to set PDF Download or Open PDF in New Tab
> * Option to choose template for pdf generation, also admin can create new templates and customize templates
> * Option to set shop name and shop address
> * Option to set footer note
> * Option set order items table header font color and background color
> * Enable/Disable Invoice PDF
> * Option to set for which order statuses, invoice pdf should be attached
> * Option to enable/disable shipping address, email address, phone number, customer notes, invoice number, invoice date in pdf invoice generation.
> * Option to enable/disable invoice download button in customer - my account page
> * Enable/Disable Packing Slip PDF
> * Option to enable/disable billing address, email address, phone number, customer notes in pdf packing slip generation.
> * Bulk download pdf invoice and packing slips from Woocommerce->Orders


> **Pro Features**
>
> * Option to set paper size for pdf
> * Admin can set shop logo image and logo height
> * Option to set automatic deletion of generated pdf files from server
> * Option to set for which Transactional Emails, invoice pdf should be attached
> * WPML Support
